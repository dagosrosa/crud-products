## Stacks utilizadas
- laravel/framework 8
- vue 2
- axios
- tailwindcss

## Clone o projeto na pasta wamp64/www
- git clone https://github.com/dagosrosa/crud-products.git

## Acesse a pasta clonada
- cd .\crud-products

## Instale as dependências
- Composer install
- npm install

## Execute o migrate para criar a tabela product
- php artisan migrate

## Rode a aplicação
- php artisan serve
